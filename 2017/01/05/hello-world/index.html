前端开发要点总结

Step1.标注

MarkMan - 个人非常推荐的一款前端工程师切图标注工具。

注意
  • 首先要划分图片模块和文字区域模块。
  • 手机端UI设计元素尺寸一般都为偶数。因为移动端双倍分辨率下所有宽、高都必须是偶数，如果设计图不符合要求需要手动调整。
  • 一个模块的位置，即x轴和y轴，必须定义一个。
  • 移动端边距主要标注比例而不是具体的值。
  • 标注文本却与要考虑行高。

Step2.切图

  • 合并图层：command + E / 按住 command 鼠标变成选区模式。
  • 另存为web格式：command+shift+alt+s。如果元素有透明的背景，则需要存在png24格式。
