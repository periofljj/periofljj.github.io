具体技术问题

CSS

  1. 解释盒子模型， box-sizing属性了解吗？
  2. 块级元素和行内元素的区别
  3. 水平居中的实现，垂直居中的实现(能说出几种实现方式)
  4. CSS3,HTML5的新特性

HTTP

  1. 常用状态码 401 403 200 500 404 302

GIT

框架

用过bootstrap，是否了解栅格化？
除了bootstrap框架是否知道自己如何实现响应式布局？

原生javascript

  1. js基本数据类型有哪些，其中undefine和null有区别吗？undefine === null?

  2. 原生js怎样判断一个对象是不是数组？在对数组的操作方法中，其中哪个方法既可以删除元素又可以替换、修改元素(splice)？

  3. 原生js对dom的操作

  4. js函数声明的几种方式？区别？

  5. js事件冒泡为概念？事件委托的适用场景？

  6. 作用域链

  7. 是否了解闭包（配题）

ES6

ES6了解过吗,promise用过吗？promise是为了解决什么问题？

思考题

已知函数式编程有两个必要特性：1.引用透明性：对于相同的输入永远都会返回同样的输出。2.没有任何可观察的副作用，也不依赖外部环境的状态。（这里的副作用包括但不限于：更改文件系统，往数据库插入记录，发送一个 http 请求，改变数据，打印/log，获取用户输入，DOM 查询，访问系统状态）。请解释下图?怎样对第一部分进行改造呢？
